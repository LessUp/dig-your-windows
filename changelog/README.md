## Format

All changelogs follow the [Keep a Changelog](https://keepachangelog.com/) format with:

- **Added** - New features
- **Changed** - Changes to existing functionality
- **Deprecated** - Soon-to-be removed features
- **Removed** - Now removed features
- **Fixed** - Bug fixes
- **Security** - Security-related fixes

## Bilingual Support

This project maintains changelogs in both English and Chinese:
- The root `CHANGELOG.md` provides a summary in English
- `changelog/CHANGELOG.zh.md` provides the complete Chinese version
- `changelog/CHANGELOG.en.md` provides the complete English version
